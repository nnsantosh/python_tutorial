## elif statement

if 1 == 2: <br/>
&nbsp;&nbsp;&nbsp;&nbsp;print("worked") <br/>
elif 3 == 3: <br/>
&nbsp;&nbsp;&nbsp;&nbsp;print('second condition true!') <br/>
else: <br/>
&nbsp;&nbsp;&nbsp;&nbsp;print('nope!') <br/>

Output will be second condition true! <br/>
